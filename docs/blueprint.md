# **App Name**: Velohub Rentals

## Core Features:

- Bike Catalog Display: A landing page presenting a grid of available bikes with title, description, and price per day.
- User Authentication: Secure login and registration functionality for users leveraging Firebase Authentication.
- Booking System: Enable users to select rental dates using a date range picker, automatically calculating the total rental price.
- Firestore Data Management: Manage and securely store all bike inventory and user booking data using Google Firestore.
- Checkout Summary: A dedicated '/checkout' page displaying a comprehensive summary of the rental details before payment.
- Payment Gateway Placeholder: Integrate a placeholder component for PayPal Smart Buttons to simulate payment processing.
- AI-Assisted Bike Descriptions: An AI tool that generates concise and appealing descriptions for each bike, highlighting its unique selling points and features.

## Style Guidelines:

- Primary color: A vibrant, clear blue (#147AEB) reflecting a modern, technical aesthetic, standing out against dark backgrounds.
- Background color: A deep, desaturated grey-blue (#141C1F) to establish a dark-themed, industrial ambiance.
- Accent color: A brighter, greenish-cyan (#8CE6DB) used for interactive elements and key information, providing contrast and dynamism.
- All text uses 'Inter', a grotesque-style sans-serif, for a modern, objective, and highly legible appearance consistent with industrial design.
- Geometric and crisp icons with strong lines, reinforcing the technical and industrial feel. Prioritize clarity and minimalist design.
- Structured grid-based layouts, providing a clean, organized, and efficient display of information, minimizing visual clutter.
- Subtle and purposeful animations, primarily for state changes and user feedback, enhancing usability without distracting from the content.
